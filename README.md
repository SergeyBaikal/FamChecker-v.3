# FamChecker-v.3
**FamChecker v.3** — это desktop-приложение на Python с графическим интерфейсом (PyQt5), предназначенное для определения и классификации вирусов по их семействам. Программа использует встроенную базу данных вирусных семейств (на основе VMR MSL40), которая содержит информацию о типе генома, хозяине, порядке, классе и описании каждого семейства.
