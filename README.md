# FamChecker v.3
**FamChecker v.3** — это desktop-приложение на Python с графическим интерфейсом (PyQt5), предназначенное для определения и классификации вирусов по их семействам. Программа использует встроенную базу данных вирусных семейств (на основе VMR MSL41, https://ictv.global/vmr), которая содержит информацию о типе генома, хозяине, порядке, классе и описании каждого семейства.


### 1. Запуск программы
- На Linux - введите в терминале `./FamChecker_v.3_linux` 
- На Windows - двойной клик по файлу exe.

После запуска откроется главное окно программы, разделенное на три панели:
- **Левая панель** — поле для ввода названий семейств
- **Центральная панель** — таблица с результатами определения
- **Правая панель** — список семейств, отсутствующих в базе данных

### 2. Импорт данных из Virus Metadata Resource (VMR)

Для импорта актуальных данных о вирусах с официального сайта ICTV:

1. Скачайте последнюю версию VMR с сайта: https://ictv.global/vmr
2. Сохраните лист VMR MSL41 (или актуальную версию) в формате **TXT** (табуляция как разделитель)
3. В программе откройте меню **"Файл"** → **"Импорт семейств из Virus Metadata Resource"**
4. Выберите скачанный файл
5. Программа автоматически:
   - Определит кодировку файла
   - Добавит новые семейства в базу данных
   - Пропустит уже существующие записи

### 3. Определение вирусов

1. В левой панели введите названия вирусных семейств (каждое семейство с новой строки, можно скопировать целый столбец из Excel)
2. Нажмите зеленую кнопку **"Определить"**
3. Результаты появятся в центральной таблице:
   - Семейство
   - Тип генома (dsDNA, ssRNA, и т.д.)
   - Хозяин
   - Порядок (Order)
   - Класс (Class)
   - Описание (если добавлено)

### 4. Управление базой данных

#### Просмотр и редактирование
- Меню **"Файл"** → **"Просмотр базы данных/Добавить описание"**
- Откроется окно со всеми семействами
- Для каждого семейства доступны:
  - ✏️ **Редактирование** — изменение информации о семействе
  - 🗑️ **Удаление** — удаление семейства из базы
- В окне доступен **поиск** по всем полям таблицы

#### Добавление нового семейства
- Меню **"Файл"** → **"Добавить семейство"**
- Заполните обязательные поля:
  - Семейство (обязательно)
  - Тип генома (обязательно)
  - Хозяин (опционально, по умолчанию "Не указан")
  - Порядок (опционально)
  - Класс (опционально)
  - Описание (опционально)

### 5. Экспорт результатов

После выполнения определения можно:

**Копировать в буфер обмена:**
- Нажмите кнопку **"Копировать результаты"**
- Данные скопируются в формате таблицы с разделителями табуляции

**Сохранить в файл:**
- Нажмите кнопку **"Сохранить результаты"**
- Выберите расположение и имя файла
- Поддерживаемые форматы: .txt, .csv

### 6. Дополнительно

- **Очистить** — очищает все поля ввода и результаты
- **Поиск в базе данных** — в окне просмотра БД есть поле для фильтрации записей
- **О программе** — меню "Справка" содержит информацию о версии и авторе
- **virus_database.json** - именно в этом файле сохраняются добавленные семейства из VMR файла, а также ваше описание. При первом запуске создаётся автоматически. Для сброса удалите этот файл — программа создаст новый с базовой версией VMR.
- **Данные семейства отсутствуют в базе данных** — это на тот случай, если в какой-либо базе были старые названия и у вас есть после аннотации, например, Myoviridae и Podoviridae.  


Программа предоставляется "как есть" без каких-либо гарантий. Надеюсь, кому-то поможет.
